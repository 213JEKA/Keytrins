## About

**Keytrins** is a Web3 gaming ecosystem built on **TON**.

The native utility token of the ecosystem is **KTRN**.

Our first product is **TONX5MINER** — a futuristic mining game where players mine, upgrade, complete missions, invite friends and earn rewards.
