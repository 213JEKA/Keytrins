# Keytrins Media Kit

## Project

Keytrins (KTRN) is a Web3 gaming ecosystem built on TON.

## Brand Style

- Dark futuristic cyber style
- Cyan / teal and purple accents
- Black background
- Keytrins emblem and wordmark
- KTRN ecosystem theme

## Official Colors

| Color | Hex |
|---|---|
| Black | #0B0D14 |
| Purple | #7A2CF7 |
| Cyan | #00E6FF |
| White | #F2F4F7 |

## Logo Usage

Use only official Keytrins logos and brand assets.

Do not:
- modify the emblem
- change colors
- stretch the logo
- add unrelated symbols
- use unofficial versions
